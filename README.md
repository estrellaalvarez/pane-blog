# Pane Blog
Pane Blog is an app where you can create an account and post/delete blogs with images.

## Technologies Used:
- React
- Express
- Node
- MongoDB
- HTML/CSS

## Stretch Features:
- Visit user's profiles to see all of their blogs
- Post in anonymous mode
- Ability to like a blog
- Edit a blog
